🚀 نصب و اجرا | Installation & Run

پیش‌نیازها | Prerequisites
- 🟦 Node.js v18+
- 📦 npm یا yarn
- 🔗 Git

مراحل نصب | Setup Steps
bash

کلون کردن ریپو | Clone the repo
git clone https://github.com/apz-chain/apz-chain-website.git
cd apz-chain-website

نصب وابستگی‌ها | Install dependencies
npm install

اجرای محلی | Run locally
npm run dev


👉 Localhost: http://localhost:3000
